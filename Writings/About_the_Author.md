# About the Author

My name is Marcus Webb and I'm a writer. I like to write stories that scare people. Horror is my favorite kind of story because it makes you feel things.

I've always loved reading books about creepy places and mysterious things. I like Stephen King and other horror writers. When I was younger, I read a lot of scary stories before bed, which probably wasn't a good idea because it gave me nightmares.

But I kept reading them anyway.

When I got older, I decided I wanted to write my own scary stories. I wanted to create worlds that felt real but also weird and dangerous. I wanted to write about normal people in abnormal situations.

Room 412 is one of my first stories. I wrote it because I was thinking about all the strange places in the world that nobody really pays attention to. Old motels, abandoned buildings, forgotten roads. Those places have stories to tell. Some of those stories are scary.

Besides writing, I like to travel. I like to drive on long roads and see different places. I like to observe people and places and think about the stories that could happen there.

I'm still learning how to write better. I'm still writing more stories. And I hope that one day, my stories will scare people in the best way possible.

If you have any thoughts about Room 412, I would like to hear them.
